Ping Client
===========
     This sample demonstrates a client application that can ping a remote host. It displays the IP address used and the response time in milliseconds.


Sample Language Implementations 
===============================
     This sample is available in the following language implementations:
     C#, VB


To build the sample using the command prompt
============================================
     1. Open an SDK Command Prompt window and navigate to one of the language-specific subdirectories under the PingClient directory.

     2. Type msbuild PingClient<lang>.sln at the command line where <lang> is either CS or VB


To build the sample using Visual Studio 
=======================================
     1. Open Windows Explorer and navigate to one of the language-specific subdirectories under the PingClient directory.

     2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.

     3. On the Build menu, click Build Solution. 

     The application is built in the default \bin\Debug directory.


To run the sample 
=================
     1. Navigate to the directory that contains the new executable, using the command prompt or Windows Explorer.

     2. Type PingClient.exe at the command line, or double-click the icon for PingClient.exe to launch it from Windows Explorer.
